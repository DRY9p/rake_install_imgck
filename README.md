# Rake for /ImageMagick Source Installer/

Установка ImageMagick из исходного кода с использованием Rake

## Требования

- **Ruby** (версия 2.5 и выше)
- **Rake**: `gem install rake`

# Установка
```bash
  git clone https://github.com/DRY9p/rake_install_imgck
```
# Использование

## Для установки
```bash
  rake imgck_install
```

## Для удаления
```bash
  rake rm_imgck
```
